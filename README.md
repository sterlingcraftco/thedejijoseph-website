# thedejijoseph.com

Personal website for Deji Joseph — Data Analyst & Strategist.

## Tech stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Hono (backend/API server)

## Local development

Requirements: Node.js & npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Clone the repo
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the dev server
npm run dev
```

## Editing

- **Local IDE** — clone, edit, push
- **GitHub** — edit files directly via the pencil icon in the file view
- **Codespaces** — open the repo, click Code → Codespaces → New codespace

## Deployment

The site is self-hosted via Docker + Caddy. Build with:

```sh
npm run build
```
